# Orbital Escape

Orbital Escape is a 3D space survival and exploration game built using Python, PyOpenGL, and GLUT. The game features real-time movement physics, planetary gravity systems, enemy obstacles, meteor attacks, and multiple gameplay levels.

## Features

- 5 progressively difficult levels
- Real-time rocket movement and physics
- Planetary orbit and gravity simulation
- Black hole gravity mechanics
- Enemy obstacles with AI movement
- Meteor shower attacks
- Shooting and collision system
- Multiple camera modes
- Cheat mode with auto targeting
- HUD system with score, life, boost, and energy tracking
- Space environment with stars and nebula effects
- Smooth animations and particle explosions

## Controls

- `W` → Move forward
- `S` → Reverse
- `A / D` → Side movement
- `B` → Boost
- `Left Mouse` → Shoot
- `Right Mouse` → Change camera
- `C` → Toggle cheat mode
- `V` → Switch camera mode
- `R` → Restart game
